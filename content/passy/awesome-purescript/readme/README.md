## Build Tooling

*   [pulp (‚≠ê448)](https://github.com/bodil/pulp)
*   [psvm-js (‚≠ê31)](https://github.com/ThomasCrvsr/psvm-js) - PureScript Version Manager
*   [purescript-psa (‚≠ê85)](https://github.com/natefaubion/purescript-psa) - A pretty, flexible error/warning reporting frontend for `psc`
*   [grunt-purescript (‚≠ê10)](https://github.com/purescript-contrib/grunt-purescript)
*   [gulp-purescript (‚≠ê34)](https://github.com/purescript-contrib/gulp-purescript)
*   [purs-loader (‚≠ê185)](https://github.com/ethul/purs-loader) for webpack
*   [psc-package (‚≠ê226)](https://github.com/purescript/psc-package) - A package manager for PureScript based on package sets
*   [spago (‚≠ê677)](https://github.com/spacchetti/spago) - PureScript package manager and build tool powered by Dhall and package-sets

*   [purescript-prelude (‚≠ê155)](https://github.com/purescript/purescript-prelude) - The standard Prelude.
*   [purescript-batteries (‚≠ê30)](https://github.com/tfausak/purescript-batteries) - A PureScript prelude with more features.

## UI Libraries

CSS:

*   [purescript-css (‚≠ê103)](https://github.com/slamdata/purescript-css) - A clean, type-safe library for describing, manipulating and rendering CSS.

React-based:

*   [purescript-thermite (‚≠ê353)](https://github.com/paf31/purescript-thermite) - A simple wrapper for ReactJS inspired by `react-blaze`.
*   [purescript-react (‚≠ê386)](https://github.com/purescript-contrib/purescript-react) - React bindings for PureScript
*   [purescript-react-basic (‚≠ê269)](https://github.com/lumihq/purescript-react-basic) - An opinionated set of bindings to the React library, optimizing for the most basic use cases.
*   [purescript-react-basic-hooks (‚≠ê187)](https://github.com/spicydonuts/purescript-react-basic-hooks) - An alternative way to define React components using React's "hooks" APIs. Compatible with `purescript-react-basic`.
*   [purescript-pux (‚≠ê567)](https://github.com/alexmingoia/purescript-pux) - Build type-safe web apps with PureScript.
*   [purescript-spork (‚≠ê152)](https://github.com/natefaubion/purescript-spork) - Elm-like for PureScript.
*   [purescript-concur (‚≠ê254)](https://github.com/ajnsit/purescript-concur) - Concur UI framework. React backend, but can use others.
*   [purescript-presto (‚≠ê179)](https://github.com/juspay/purescript-presto) - Write Apps like Mathematical Equations!
*   [purescript-optic-ui (‚≠ê119)](https://github.com/zrho/purescript-optic-ui) - Write single page web user interfaces declaratively and concisely with the help of lenses and traversals.
*   [purescript-react-explor (‚≠ê88)](https://github.com/paf31/purescript-react-explore) - Experiments with comonads for modelling React UIs.

Virtual-DOM based:

*   [purescript-halogen (‚≠ê1.4k)](https://github.com/slamdata/purescript-halogen) - A declarative, type-safe UI library.
*   [purescript-hedwig (‚≠ê128)](https://github.com/utkarshkukreti/purescript-hedwig) - Hedwig is a fast, type safe, declarative PureScript library for building web applications.
*   [purescript-flame (‚≠ê259)](https://github.com/easafe/purescript-flame) - Flame is a fast & simple framework for building web applications in PureScript inspired by purescript-hedwig and Elm

Signals/Behaviors/FRP:

*   [purescript-flare (‚≠ê285)](https://github.com/sharkdp/purescript-flare) - Applicative-style reactive user interfaces built on top of purescript-signal.
*   [purescript-turbine (‚≠ê80)](https://github.com/funkia/purescript-turbine) - Purely functional frontend framework powered by FRP.
*   [purescript-outwatch (‚≠ê33)](https://github.com/OutWatch/purescript-outwatch) - A functional and reactive UI framework based on Rx and VirtualDom.
*   [purescript-panda (‚≠ê68)](https://github.com/i-am-tom/purescript-panda) - What would TEA look like if we had no VDOM?
*   [purescript-specular (‚≠ê119)](https://github.com/restaumatic/purescript-specular) - A Reflex-Dom inspired UI library for PureScript.
*   [purescript-sdom (‚≠ê161)](https://github.com/paf31/purescript-sdom) - An experiment in replacing the virtual DOM and avoiding diffing.
*   [purescript-purview (‚≠ê110)](https://github.com/paf31/purescript-purview) - A UI library based on the incremental lambda calculus.

## URL Routers

*   [purescript-routing (‚≠ê95)](https://github.com/slamdata/purescript-routing) - A clean, type-safe routing library for PureScript.
*   [purescript-cofree-react-router (‚≠ê24)](https://github.com/coot/purescript-cofree-react-router) - React-Router implemented in PureScript as Cofree Comonad for the Array functor.
*   [purescript-trout (‚≠ê42)](https://github.com/owickstrom/purescript-trout) - Type-level routing for PureScript.
*   [purescript-boomboom (‚≠ê5)](https://github.com/paluh/purescript-boomboom) - Never hard code your urls again. Boomboom them all!
*   [purescript-routing-duplex (‚≠ê79)](https://github.com/natefaubion/purescript-routing-duplex) - Simple bidirectional parser/printers for your routing data types.

## Components

*   [purescript-halogen-formless (‚≠ê125)](https://github.com/thomashoneyman/purescript-halogen-formless) - A renderless component for building painless forms in Halogen
*   [purescript-halogen-select (‚≠ê63)](https://github.com/citizennet/purescript-halogen-select) - Building blocks for common selection user interfaces like dropdowns, typeaheads, image pickers, and calendars.
*   [purescript-halogen-echarts (‚≠ê8)](https://github.com/slamdata/purescript-halogen-echarts) - A Halogen integration for the popular ECharts charting library
*   [purescript-halogen-day-picker (‚≠ê13)](https://github.com/rnons/purescript-halogen-day-picker) - A day picker for Halogen
*   [purescript-halogen-menu (‚≠ê8)](https://github.com/slamdata/purescript-halogen-menu) - Interactive menus in Halogen

## Asynchronicity and Parallelism

*   [purescript-aff (‚≠ê271)](https://github.com/slamdata/purescript-aff) - An asynchronous effect monad for PureScript.
*   [purescript-affjax (‚≠ê271)](https://github.com/slamdata/purescript-aff) - An asynchronous AJAX library built using Aff.
*   [purescript-queue (‚≠ê3)](https://github.com/athanclark/purescript-queue) - An asynchronous queue library for PureScript.
*   [purescript-signal (‚≠ê251)](https://github.com/bodil/purescript-signal) - An asynchronous signal library for PureScript, similar to Elm.
*   [purescript-zeta (‚≠ê5)](https://github.com/athanclark/purescript-zeta) - An alternative signal implementation in PureScript.

## Effect Management

*   [purescript-eff (‚≠ê38)](https://github.com/purescript/purescript-eff) - The Eff monad, for handling native side effects.
*   [purescript-effect (‚≠ê49)](https://github.com/purescript/purescript-effect) - The Effect monad, for handling native side effects in PureScript v0.12.x.

## Json Serialization

*   [purescript-argonaut (‚≠ê59)](https://github.com/purescript-contrib/purescript-argonaut) - Json encoding and decoding typeclasses.
*   [purescript-simple-json (‚≠ê131)](https://github.com/justinwoo/purescript-simple-json) - Json encoding and decoding through Generics.

## Binary Serialization

*   [purescript-arraybuffer (‚≠ê6)](https://github.com/jacereda/purescript-arraybuffer) - Bindings to the ArrayBuffer JavaScript type.
*   [purescript-arraybuffer-class (‚≠ê5)](https://github.com/athanclark/purescript-arraybuffer-class) - Typeclasses for ArrayBuffer encoding and decoding.
*   [purescript-arraybuffer-builder (‚≠ê4)](https://github.com/jamesdbrock/purescript-arraybuffer-builder) - Builder for serializing ArrayBuffer
*   [purescript-parsing-dataview (‚≠ê3)](https://github.com/jamesdbrock/purescript-parsing-dataview) - Parser for deserializing ArrayBuffer
*   [purescript-protobuf (‚≠ê47)](https://github.com/xc-jp/purescript-protobuf) - Google Protocol Buffers
