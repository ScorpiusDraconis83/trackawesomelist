## Bindings

*   [RxAndroid (‚≠ê20k)](https://github.com/ReactiveX/RxAndroid) - Android specific bindings for RxJava.
*   [RxBinding (‚≠ê9.7k)](https://github.com/JakeWharton/RxBinding) - RxJava binding APIs for Android UI widgets from the platform and support libraries.
*   [rx-preferences (‚≠ê1.5k)](https://github.com/f2prateek/rx-preferences) - Reactive `SharedPreferences` for Android.
*   [RxPermissions (‚≠ê10k)](https://github.com/tbruyelle/RxPermissions) - Android M runtime permissions powered by RxJava.
*   [SQLBrite (‚≠ê4.6k)](https://github.com/square/sqlbrite) - A lightweight wrapper around SQLiteOpenHelper and ContentResolver which introduces reactive stream semantics to queries.
*   [Android-ReactiveLocation (‚≠ê2.1k)](https://github.com/mcharmas/Android-ReactiveLocation) - Small library that wraps Google Play Service API in brilliant RxJava Observables reducing boilerplate to minimum.
*   [ReactiveNetwork (‚≠ê2.5k)](https://github.com/pwittchen/ReactiveNetwork) - Android library listening network connection state and change of the WiFi signal strength with RxJava Observables.
*   [ReactiveSensors (‚≠ê165)](https://github.com/pwittchen/ReactiveSensors) - Android library monitoring hardware sensors with RxJava Observables.
*   [RxPalette (‚≠ê201)](https://github.com/hzsweers/RxPalette) - RxJava bindings for the Palette library on Android.
*   [rxjava-jdbc (‚≠ê804)](https://github.com/davidmoten/rxjava-jdbc) - Efficient execution and functional composition of database calls using jdbc and RxJava Observables.
*   [rxjava-file (‚≠ê82)](https://github.com/davidmoten/rxjava-file) - RxJava observables for files including NIO events.
*   [RxTuples (‚≠ê113)](https://github.com/pakoito/RxTuples) - Simple tuples to use with RxJava.
*   [RxAnimationBinding (‚≠ê83)](https://github.com/blipinsk/RxAnimationBinding) - RxJava binding APIs for Android's animations.
