A collection of awesome [Refinery CMS](http://www.refinerycms.com/) extensions, resources and shiny things.

## Frontoffice extensions

*   [refinerycms-copywriting (‚≠ê51)](https://github.com/unixcharles/refinerycms-copywriting) - Write once, display at the infinite
*   [refinerycms-elasticsearch (‚≠ê6)](https://github.com/refinerycms-contrib/refinerycms-elasticsearch) - Fulltext search with [Elasticsearch (‚≠ê1.9k)](https://github.com/elastic/elasticsearch-ruby)
*   [refinerycms-image-slideshows (‚≠ê8)](https://github.com/bisscomm/refinerycms-image-slideshows) - Display slideshows
*   [refinerycms-jobs (‚≠ê5)](https://github.com/bisscomm/refinerycms-jobs) - Job Posting and Job Applications
*   [refinerycms-products (‚≠ê7)](https://github.com/bisscomm/refinerycms-products) - Display products
*   [refinerycms-retailers (‚≠ê3)](https://github.com/bisscomm/refinerycms-retailers) - Display retailers
*   [refinerycms-teams (‚≠ê4)](https://github.com/bisscomm/refinerycms-teams) - Display teams

*   [refinerycms-dynamicfields (‚≠ê16)](https://github.com/jfalameda/refinerycms-dynamicfields) - Create models dynamically and attach them to pages
*   [refinerycms-page-resources (‚≠ê6)](https://github.com/anitagraham/refinerycms-page-resources) - Attach resources (files) to a Page
*   [refinerycms-redactor (‚≠ê2)](https://github.com/rabid/refinerycms-redactor) - WYSIWYG editor with [Redactor](https://imperavi.com/redactor/)

*   [refinerycms-snoods (‚≠ê1)](https://github.com/cleverlemming/refinerycms-snoods) - Creating download pages and sending leads to Salesforce
*   [refinerycms-tinymce (‚≠ê7)](https://github.com/ghoppe/refinerycms-tinymce) - TinyMCE Visual Editing
