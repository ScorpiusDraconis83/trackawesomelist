## Modules

### Array

*   [is-sorted (‚≠ê21)](https://github.com/dcousens/is-sorted) - A small module to check if an Array is sorted.
*   [array-first (‚≠ê11)](https://github.com/jonschlinkert/array-first) - Get the first element or first n elements of an array.
*   [array-last (‚≠ê34)](https://github.com/jonschlinkert/array-last) - Return the last element in an array.
*   [arr-flatten (‚≠ê61)](https://github.com/jonschlinkert/arr-flatten) - Recursively flatten an array or arrays.
*   [dedupe (‚≠ê24)](https://github.com/seriousManual/dedupe) - Remove duplicates from an array.
*   [array-range (‚≠ê25)](https://github.com/mattdesl/array-range) - Creates a new array with given range.
*   [arr-diff (‚≠ê45)](https://github.com/jonschlinkert/arr-diff) - Returns an array with only the unique values from the first array, by excluding all values from additional arrays using strict equality for comparisons.
*   [filled-array (‚≠ê41)](https://github.com/sindresorhus/filled-array) - Returns an array filled with the specified input
*   [map-array (‚≠ê4)](https://github.com/parro-it/map-array) - Map object keys and values into an array.
*   [in-array (‚≠ê16)](https://github.com/jonschlinkert/in-array) - Return true if any of passed values exists in array - faster than using indexOf.
*   [unordered-array-remove (‚≠ê118)](https://github.com/mafintosh/unordered-array-remove) - Efficiently remove an element from an unordered array without doing a splice.
*   [array-swap (‚≠ê7)](https://github.com/michaelzoidl/swap-array) - Swap position of two items in an array.
*   [mirrarray (‚≠ê5)](https://github.com/johnwquarles/mirrarray) - Creates a keymirror object from an array of valid keys.
*   [group-array (‚≠ê53)](https://github.com/doowb/group-array) - Group array of objects into lists.
*   [array.chunk (‚≠ê11)](https://github.com/zhiyelee/array.chunk) - Split array/TypedArray to chunks of given size.
*   [fast-cartesian (‚≠ê54)](https://github.com/ehmicky/fast-cartesian) - Fast cartesian product.

### String

*   [decamelize (‚≠ê226)](https://github.com/sindresorhus/decamelize) - Convert a camelized string into a lowercased one with a custom separator: unicornRainbow ‚Üí unicorn\_rainbow.
*   [pad-left (‚≠ê44)](https://github.com/jonschlinkert/pad-left) - Left pad a string with zeros or a specified string.
*   [to-camel-case (‚≠ê36)](https://github.com/ianstormtaylor/to-camel-case) - Convert a string to a camel case.
*   [to-capital-case (‚≠ê11)](https://github.com/ianstormtaylor/to-capital-case) - Convert a string to a capital case.
*   [to-constant-case (‚≠ê4)](https://github.com/ianstormtaylor/to-constant-case) - Convert a string to a constant case.
*   [to-dot-case (‚≠ê5)](https://github.com/ianstormtaylor/to-dot-case) - Convert a string to a dot case.
*   [to-no-case (‚≠ê16)](https://github.com/ianstormtaylor/to-no-case) - Remove an existing case from a string.
*   [to-pascal-case (‚≠ê3)](https://github.com/ianstormtaylor/to-pascal-case) - Convert a string to a pascal case.
*   [to-sentence-case (‚≠ê6)](https://github.com/ianstormtaylor/to-sentence-case) - Convert a string to a sentence case.
*   [to-snake-case (‚≠ê26)](https://github.com/ianstormtaylor/to-snake-case) - Convert a string to a snake case.
*   [to-space-case (‚≠ê5)](https://github.com/ianstormtaylor/to-space-case) - Convert a string to a space case.
*   [to-title-case (‚≠ê17)](https://github.com/ianstormtaylor/to-title-case) - Convert a string to a title case.
*   [node-slug (‚≠ê1.1k)](https://github.com/dodo/node-slug) - slugifies even utf-8 chars.
*   [rtrim (‚≠ê0)](https://github.com/sergejmueller/rtrim) - Strip whitespace - or other characters - from the end of a string.
*   [slice.js (‚≠ê34)](https://github.com/hustcc/slice.js) - Javascript library to enhance String.substring / Array.slice with python slice style.
*   [strip-ansi (‚≠ê359)](https://github.com/chalk/strip-ansi) - Strip ANSI escape codes.
*   [striptags (‚≠ê481)](https://github.com/ericnorris/striptags) - An implementation of PHP's strip\_tags in Node.js.
*   [parse-next-json-value (‚≠ê1)](https://github.com/ErikOnBike/parse-next-json-value) - Parse next JSON value from string allowing extraneous characters after value.
*   [pluralize (‚≠ê1)](https://github.com/DaniAkash/pluralizer) - A very tiny library to pluralize words

### Date & Time

*   [pretty-ms (‚≠ê924)](https://github.com/sindresorhus/pretty-ms) - Convert milliseconds to a human readable string: 1337000000 ‚Üí 15d 11h 23m 20s.
*   [hirestime (‚≠ê16)](https://github.com/seriousManual/hirestime) - A wrapper around the built-in high resolution timer which simplifies the calculation of timestamps.
*   [periods (‚≠ê1)](https://github.com/timruffles/periods) - Defined time-periods constants for Javascript, in milliseconds.
*   [fecha (‚≠ê2k)](https://github.com/taylorhakes/fecha) - Javascript Date formatting and parsing.
*   [akamai-time-reference (‚≠ê3)](https://github.com/jucrouzet/akamai-time-reference) - Get reference time using Akamai's time reference service.
*   [timeago.js (‚≠ê5k)](https://github.com/hustcc/timeago.js) - A tiny(\~1.7kb) library used to format date with `*** time ago` statement.
*   [count-days-in-month (‚≠ê1)](https://github.com/shinnn/count-days-in-month) - Get the number of days in a given month.
*   [time-stamp (‚≠ê111)](https://github.com/jonschlinkert/time-stamp) - Get a formatted timestamp.
*   [twas (‚≠ê244)](https://github.com/vutran/twas) - Generate a relative time string (Example: "3 seconds ago")

### Object

*   [map-obj (‚≠ê173)](https://github.com/sindresorhus/map-obj) - Map object keys and values into a new object.
*   [filter-obj (‚≠ê89)](https://github.com/sindresorhus/filter-obj) - Filter object keys and values into a new object.
*   [object-values (‚≠ê23)](https://github.com/sindresorhus/object-values) - Get the values of an object.
*   [object-pairs (‚≠ê2)](https://github.com/eush77/object-pairs) - Turn an object into list of \[key, value] pairs for mapping, iterating or other purposes.
*   [zipmap (‚≠ê4)](https://github.com/landau/zipmap) - Returns a map with the keys mapped to the corresponding vals. zipmap also accepts a single value of objects or pairs.
*   [just-pluck (‚≠ê11)](https://github.com/jarofghosts/just-pluck) - Pluck without the madness.
*   [deep-equal (‚≠ê729)](https://github.com/substack/node-deep-equal) - Node's assert.deepEqual() algorithm as a standalone module.
*   [deep-assign (‚≠ê249)](https://github.com/sindresorhus/deep-assign) - Recursive Object.assign().
*   [set-value (‚≠ê260)](https://github.com/jonschlinkert/set-value) - Create nested values and any intermediaries dot notation (`'a.b.c'`) paths.
*   [get-value (‚≠ê223)](https://github.com/jonschlinkert/get-value) - Use property paths (a.b.c) to get a nested value from an object.
*   [has-value (‚≠ê27)](https://github.com/jonschlinkert/has-value) - Returns true if a value exists, false if empty. Works with deeply nested values using dot notation (`'a.b.c'`) paths.
*   [has-key-deep (‚≠ê1)](https://github.com/ryanaghdam/has-key-deep) - Deep-search objects for keys. Keys can be searched by providing an array of keys, or using a dot-notiation.
*   [flatkeys (‚≠ê3)](https://github.com/ricardobeat/flatkeys) - Flatten object key hierarchies into a list of strings using a custom separator.
*   [flatten-obj (‚≠ê20)](https://github.com/watson/flatten-obj) - Converts an object literal with deeply nested nodes to a simple key/value object.
*   [is-empty-object (‚≠ê11)](https://github.com/gummesson/is-empty-object) - Check if an object is empty.
*   [stringify-object (‚≠ê297)](https://github.com/yeoman/stringify-object) - Stringify an object/array like JSON.stringify just without all the double-quotes.
*   [sorted-object (‚≠ê35)](https://github.com/domenic/sorted-object) - Returns a copy of an object with its keys sorted.
*   [static-props (‚≠ê3)](https://github.com/fibo/static-props) - Defines static object attributes using `Object.defineProperties`
*   [missing-deep-keys (‚≠ê0)](https://github.com/vladgolubev/missing-deep-keys) - Returns an array of keys from first object that are missing in second.
*   [has-own-property (‚≠ê2)](https://github.com/LinusU/has-own-property) - Check if an object has a local property.
*   [merge-objects (‚≠ê2)](https://github.com/shevaroller/node-merge-objects) - Deep-merge two objects. Arrays that are values of the same object key get concatenated.
*   [deep-object-diff (‚≠ê926)](https://github.com/mattphillips/deep-object-diff) - Deep diff two JavaScript Objects while preserving the data structure. Including nested structures of Arrays and Objects.

### Function

*   [compose-function (‚≠ê57)](https://github.com/stoeffel/compose-function) - Compose a new function from smaller functions `f(g(x))`.
*   [curry (‚≠ê314)](https://github.com/dominictarr/curry) - A curry function without anything too clever.
*   [once (‚≠ê215)](https://github.com/isaacs/once) - Run a function exactly one time.
*   [deep-bind (‚≠ê8)](https://github.com/jonschlinkert/deep-bind) - Bind a context to all functions in an object, including deeply nested functions.
*   [identity-function](https://github.com/substack/identity-function) - Always return the input argument.
*   [mem (‚≠ê1k)](https://github.com/sindresorhus/mem) - An optimization technique used to speed up consecutive function calls by caching the result of calls with identical input.
*   [throttle-debounce (‚≠ê958)](https://github.com/niksy/throttle-debounce) - Throttle/debounce your functions.
*   [compose-tiny (‚≠ê46)](https://github.com/hipstersmoothie/compose-tiny) - A very tiny and fast compose function.

### Math

*   [is-even (‚≠ê179)](https://github.com/jonschlinkert/is-even) - A good way to tell if a number is even or not (avoids type issues). Uses `is-odd` and `is-number` under the hood.
*   [is-number (‚≠ê229)](https://github.com/jonschlinkert/is-number) - Returns `true` if the value is a number.
*   [is-odd (‚≠ê268)](https://github.com/jonschlinkert/is-odd) - A good way to tell if a number is odd or not (avoids type issues). Uses `is-number` under the hood.
*   [easy-math.js (‚≠ê7)](https://github.com/kingzez/easy-math.js) - A tiny easy math library including addition, multiplication, subtraction, and division.
*   [my-prime (‚≠ê3)](https://github.com/jinnatul/my-prime) - A good way to tell if a number is prime or not.
*   [fun-gcd (‚≠ê1)](https://github.com/zubayerhimel/fun-gcd) - A tiny math library to get gcd of two numbers using Euclidean algorithm

### Stream

*   [through2 (‚≠ê1.9k)](https://github.com/rvagg/through2) - Tiny wrapper around Node streams2 Transform to avoid explicit subclassing noise.
*   [through2-filter (‚≠ê35)](https://github.com/brycebaril/through2-filter) - A through2 to create an Array.prototype.filter analog for streams.
*   [through2-map (‚≠ê70)](https://github.com/brycebaril/through2-map) - A through2 to create an Array.prototype.map analog for streams.
*   [stream-spigot (‚≠ê15)](https://github.com/brycebaril/node-stream-spigot) - A readable stream generator, useful for testing or converting simple functions into Readable streams.
*   [concat-stream (‚≠ê569)](https://github.com/maxogden/concat-stream) - writable stream that concatenates strings or data and calls a callback with the result.
*   [JSONStream (‚≠ê1.9k)](https://github.com/dominictarr/JSONStream) - streaming JSON.parse and stringify
*   [through2-map-promise (‚≠ê3)](https://github.com/RangerMauve/through2-map-promise) - A small promise-based wrapper for through2.
*   [pump (‚≠ê897)](https://github.com/mafintosh/pump) - pipe streams together and close all of them if one of them closes.
*   [split (‚≠ê346)](https://github.com/dominictarr/split) - Break up a stream and reassemble it so that each line is a chunk.
*   [is-stream (‚≠ê106)](https://github.com/sindresorhus/is-stream) - Check if something is a Node.js stream.
*   [syncthrough (‚≠ê77)](https://github.com/mcollina/syncthrough) - Transform your data as it pass by, synchronously.

### Promise

*   [pify (‚≠ê1.5k)](https://github.com/sindresorhus/pify) - Promisify a callback-style function.
*   [promise-all-props (‚≠ê11)](https://github.com/Siilwyn/promise-all-props) - Like `Promise.all` but for object properties.
*   [sleep-promise (‚≠ê81)](https://github.com/brummelte/sleep-promise) - Resolves a promise after a specified delay.
*   [is-promise (‚≠ê278)](https://github.com/then/is-promise) - Test whether an object looks like a promises-a+ promise.
